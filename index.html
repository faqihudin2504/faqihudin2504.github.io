  • Web Single Page

    Web single page adalah web yang hanya memiliki satu halaman saja,
    sehingga ketika pengguna mengklik salah satu elemen, maka web tersebut tidak akan me-reload seluruh halaman web tersebut.

  • Web Multipage

    Web multipage adalah web yang memiliki lebih dari satu halaman,
    ketika pengguna mengklik salah satu elemen, maka website tersebut akan me-reload seluruh elemen dari halaman web tersebut.
